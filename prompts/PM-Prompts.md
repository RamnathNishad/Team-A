# PM-Prompts.md

## Project Management Prompts for SmartLoan

### Project Overview
This document contains prompts and guidelines for Project Managers working on the SmartLoan capstone project.

### Sprint Planning
- Define user stories for each sprint
- Estimate story points using Fibonacci sequence
- Allocate resources based on team capacity
- Schedule sprint reviews and retros

### Stakeholder Communication
- Weekly status updates
- Risk assessment and mitigation
- Timeline tracking and adjustments
- Budget monitoring

### Deliverables Tracking
- Frontend: React.js/Next.js application
- Backend: ASP.NET Core REST API
- Database: SQL Server implementation
- Documentation: Complete technical and user documentation

### Risk Management
- Technical risks: Framework compatibility, performance issues
- Resource risks: Team availability, skill gaps
- Timeline risks: Scope creep, dependency delays
